# BenchPress

An OOP framework for building modern Wordpress themes and plugins

## Dependencies

1. Docker Desktop
2. Docker Compose
3. Composer
4. Node
5. NPM or Yarn

## Local Development

#### Clone the repo

#### Install Dependencies
- `composer update`
- `cd ./theme && npm i && cd..`

#### Create .env
- `cp .env.example .env`
- ##### Edit .env to change any varibales as you prefer

#### Run with Docker Compose
- `docker compse up -d`
